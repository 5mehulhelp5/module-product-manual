# Magento 2 Product Manual

Magento 2 module that adds an Assembly Manual PDF upload field to products.

## Features

- Upload PDF from Product Edit
- Save file under pub/media/product/manuals
- Show Manual on Product Detail Page
- Supports custom icon
- Compatible with Magento 2.4.x

## Installation

```
app/code/GDMexico/ProductManual
```

Run

```
bin/magento setup:upgrade
bin/magento cache:flush
```
